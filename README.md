# Blockchain-Enabled Secure Stock Recommendation System

## Overview

The Blockchain-Enabled Secure Stock Recommendation System is a web application that provides secure and transparent stock recommendations using blockchain technology. The platform helps users analyze stock information, view recommendations, and make informed investment decisions while ensuring data integrity and transparency.

## Features

* Secure stock recommendation system
* Blockchain-based verification
* User authentication (Login/Signup)
* Stock search functionality
* Personalized recommendations
* Dashboard with stock insights
* Responsive user interface

## Installation

### Clone the Repository

```bash
git clone https://github.com/DarshanPatil45/Blockchain-Enabled-Secure-Stock-Recommendation-System.git
```

### Navigate to Project Directory

```bash
cd Blockchain-Enabled-Secure-Stock-Recommendation-System
```

### Install Dependencies

```bash
npm install
```

## Run the Application

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```

## Build for Production

```bash
npm run build
```
